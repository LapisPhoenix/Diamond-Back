# Installation

1. Clone this repo
   ```bash
   git clone https://github.com/LapisPhoenix/Diamond-Back.git
   ```
2. CD into `Diamond-Back`
   ```bash
   cd Diamond-Back
   ```
3. Install Depenecies
   ```bash
   pip install -r requirements.txt
   ```
